# Multi Page Website

This project is a personal portfolio website created using HTML and CSS.  
It is built as a continuation of the previous Flexbox layout task and focuses on organizing content across multiple pages while maintaining a consistent design.

## Pages Included
- Home
- About
- Skills
- Contact

## Key Features
- Multi-page website with consistent navigation
- Reusable external CSS file
- Responsive layout using Flexbox
- Active navigation highlight for the current page
- Clean and simple structure using semantic HTML

## Technologies Used
- HTML5
- CSS3
- Flexbox

## Purpose
The purpose of this project is to demonstrate the ability to structure a multi-page website, link pages using relative paths, and maintain a consistent layout without changing the original design.
